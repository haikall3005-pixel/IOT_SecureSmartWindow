# Secure Smart Window System

## Project Overview

A smart home security system developed using ESP32 and Node-RED.

## Features

- Temperature Monitoring
- Humidity Monitoring
- Motion Detection
- Rain Detection
- Window Status Detection
- Real-Time Dashboard
- Motion Alarm (Buzzer)

## Hardware

- ESP32
- DHT11
- PIR Sensor
- Rain Sensor
- Reed Switch
- LCD I2C
- Active Buzzer

## Software

- Arduino IDE
- Node-RED
- GitHub

## Architecture

ESP32 → Node-RED Dashboard
